# bao · the shell

bao is the shell — the bridge where you talk to the ship. the live image below *is* bao, in
your browser: the very bytes that boot on bare metal, compiled to wasm. type an expression and
press enter — each line is one expression (infix works), an open paren continues to the next
line, and up/down walk history. errors print the condition data and the session keeps going.
