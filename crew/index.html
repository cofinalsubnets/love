the crew

ai is sailed by a small crew — ten little programs, each with one thing it tends, each written in ai over the same tiny core. they ride kship, the freestanding kernel (a ship in port); non-overlapping territory, so the sessions run in parallel without colliding.

ai · the pilot
the mind at the helm — the decide in perceive→decide→act, the policy that answers which way.
siri · the synthesist
keeps the human words and the ai words saying the same thing, so everything stays green.
sift · the garbage collector
tends the two-space copying collector — the litter box — and keeps it clean.
bao · the shell
one editor and the condition system — a soft wrap around a chewy command; also the debugger.
aineko · the cat
a netcat clone (愛猫, beloved cat) — bytes pumped both ways over a socket.
cook · the builder
make-in-ai — reads a Makefile, builds the host from scratch, passes the corpus.
tele · the eye
a telescope that scopes constellations — tensors and reverse-mode autograd over the galaxy.
zev · the reader
a parser-combinator library — tiny functions that compose a string into a tree.
wev · the weaver
a partial evaluator — give it some inputs, it weaves back a smaller program.
kship · the ship
the freestanding kernel itself — the language perceiving, deciding, acting on bare metal.

each name links to its bio; the full agent briefs are the crew/<name>.md files in the repo.
