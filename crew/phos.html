phos

mantis shrimp ◦ the window manager ◦ frequents a brine pool in holo's cave

phos is an xmonad clone written in ai (crew/phos/), speaking the raw X11 wire protocol — no Xlib, no XCB, one socket nif. make install ships bin/phos; mod+q restarts it in place, adopting the windows that survive.

the shape

  • core.l — the pure StackSet: zipper + sheaf + floats. no X, no sockets
  • layout.l — a layout is a closure, a message is a symbol
  • wire.l — the X11 codec: requests, the event/reply demux, atoms
  • manage.l + keys.l + ewmh.l — the manageHook DSL, bindings as data, the _NET_* protocol
  • config.l — what the user edits

the pure core needs no nifs, so it rides the test corpus on every target with xmonad's QuickCheck laws as asserts. where Haskell needs existential types to stack layouts, a layout here is just rect → row → placements and () means "not my message" — the fallthrough is free.

the proved part

a mantis shrimp sees more colors than you. sigs.l is an optional static overlay — observe-only, erasable — that checks the core's boundaries, and tools/wm2uu.l compiles the core's operations into a dependent type theory (test/uu.l), where Swierstra's insert invariant — a fresh window into a duplicate-free row stays duplicate-free — is proved of the compiled terms and certified axiom-free by both Rocq and Lean. break the insert and make test rejects the proof.
