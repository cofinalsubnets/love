🎨 language



love (aka ai, fka gwen lisp) is a lisp superset where every value is a one-argument
function. functions are curried, punctuation symbols are applied infix, and many parens
can be omitted, so it feels kind of like haskell, just eagerly evaluated and dynamically
typed. all primitives are generic so pretty much every paren balanced expression evaluates
to something...  unless you infinite loop 😨


the api is currently 🫨 unstable.
type names () at the repl to see a full list of the namespace.


🎯 global function type


literally everything is a one-argument function, including numbers (iterators),
collections (accessors), and multi-argument functions (curried).


🔣 (extra lisp syntax)


love uses a relaxed and simplified form of s-expressions with
three special forms, : let ? cond \ lambda.
love also has optional infix and prefix notation, which desugar to lisp. when
subexpressions are grouped by infix operators, their own parens may be omitted. for
example, (2 ** 3 = 3 ** 2 - 1 ? 'y 'n) is equivalent to
(? (= (** 2 3) (- (** 3 2) 1)) 'y 'n) (answer is y).

🚨 differences from other lisp


no thunks


since every function takes one argument,
love has no zero-argument functions or thunks.
a singleton list has the value of its element.
if you want a thunk, write a normal function and ignore the argument.


no varargs


kind of a dual situation, except there are multiple ways to implement varargs,
in addition to simulating them with macros. however, they're not built in and
don't automatically construct a list for you.



glossary



  
📋 apply

    
(verb) the binary operation of calling a function with an argument. corresponds to exponentiation in church arithmetic. written (f x) meaning apply f to x. see also: curry

  
💥 big

    
a big integer aka bignum. a type of star

  
🔵 blue

    
color of a value whose net is 0: (0, (), "", '(1 2 -3)). boolean false.

  
📒 book

    
aka tablet. a mutable hash table

  
📦 box

  
a book used as a mutable container for another value

  
🧢 cap

  
contents of the above part of the product. aka car/fst/hd. see also: cup chain link

  
⛓️ chain

  
a free binary product, aka pair or cons cell. used for lists per lisp usual. see also: link cap cup.

  
🧿 charm

    
a register-allocated fixed-width integer, aka fixnum. a type of star

  
⛪ church numeral

    
natural numbers interpreted as iterators. for example, (0 x) = 1, (1 x) = x, (2 x y) = x (x y), etc.
      numeral-on-numeral application corresponds to exponentiation. numeral composition corresponds to multiplication.

  
🪙 coin

    
a value of a newtype. see also: die

  
🌈 color

    
the order-sign of a value's net: red, green or blue

  
✨ constellation

  
any numeric type. a star or galaxy

  
🧋 cup

  
contents of the underneath part of the product. aka cdr/snd/tl. see also: cap chain link

  
🍛 curry

    
(verb) to unroll a multi-argument function into a sequence of one-argument functions. see also: apply

  
🎲 die

    
aka a newtype. see also: coin

  
🌌 galaxy

    
a tray of stars, aka shaped numeric array

  
💎 gem

    
a floating point number (double or float). a type of star

  
🟢 green

    
color of a value whose net is positive: 1 "hi" '(2 3 -1) (\ x x). boolean true.

  
🔥 hot

  
an opaque internal or external runtime object. conceptually and positionally dual to mints on the lattice

  
🃏 kind

    
a value's type classification under the lattice

  
🔷 lattice

    
the runtime-reflective partial order over all values that guides generic dispatch and comparison

  
🔗 link

    
the chain constructor, aka cons/(,)

  
🍬 mint

  
an otherwise empty unique identifier, aka gensym. conceptually, the non-zero mints fill a plane
    directly above () in the
    lattice.

  
🪪 name

    
aka a lisp symbol or erlang atom.
      formally considered as a die over a chain of string and mint.
    inherits algebra from strings.

  
🌐 net (+)

    
a value's content measure, a star

  
🈚 nil

  
the initial mint, acting as a global empty element (). blue, algebraically neutral, conceptually the lowest inhabited point of the lattice.

  
🔴 red

    
color of a value whose net is negative: -1 '(1 2 3 -99999). boolean false.

  
🌊 saturate ($)

  
retraction of a value's net (a star-typed value)
    onto the blue-green
    charms, ie. an integer in the interval [0, max-charm].
    overflow and underflow go to the limit instead of wrapping (saturation).
    thus green
    saturates to green
    while both red
    and blue
    saturate to blue.

  
🌟 star

    
a scalar fixpoint of net: charm, sun, big, gem or twin.

  
🧵 string

    
a byte vector: indexes its bytes, nets its charm-sum.

  
☀️ sun

    
a pointer-width integer (intptr_t). a type of star

  
📱 tablet

    
see: book

  
🧮 tally

    
surface level item count. length of a chain, string, tray or book

  
📥 tray

    
a shaped array. see also chain, galaxy

  
⚖️ truth value

    
given by the color of a value's saturated net:
    blue false, green true.

  
💠 twin

    
a gem-based complex number, written ~(re im). a type of star

  
1️⃣ unit

    
see: nil

  
0️⃣ zero point

    
see: nil
