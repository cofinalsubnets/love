ai would rather be felt than seen.

ai is a lisp / ml / apl hybrid over a tiny portable C core. every value is a total unary function, so application is always defined, and almost no expression is an error.

application. every value is a one-argument function, and application curries -- that is the whole language. (f) is f, (f x y) is ((f x) y), so partial application is just leaving arguments off the end. three special forms do the rest: : binds and sequences (letrec*, the last form its value), ? is cond (test/result pairs, then an else), and \ is lambda -- handed a single operand it has nothing left to bind, so it returns it unevaluated and quote falls out, 'x is (\ x).
net. every value has a net -- one complex number summed from its parts: a number is itself, a string its bytes, a list its spine, opposite phases cancelling as vectors. the sign of the net is the value's color -- green nonnegative, red negative, blue zero. the $ operator (saturate) pulls any value through its net onto the green charms, sending red values to blue. this defines truth value in conditionals, so in ai false means "negative or empty".
ring. + and * are two generic operations, total across every kind, and + is the net made total. () -- nothing -- is their shared unit, doing nothing everywhere: the two-sided identity of both (() + x and () * x are x), false in every test, const-1 when applied -- never the number 0, which annihilates *. together they form a ring that only strengthens along the lattice (a free monoid on strings, a semiring on lists, a ring on numbers), and the lattice is open: a die strikes your own coin, a value family riding the same dispatch with () for nothing.
notation. ai is a lisp, with two notations layered over the parentheses: application binds to the left, operators to the right. a call is ml-style and curried, read leftward -- (f x y) is ((f x) y). operators are apl-style and have no precedence, read rightward -- 1 + 2 * 3 is 1 + (2 * 3). and every operator is secretly two: spaced is dyadic, glued is monadic -- a - b subtracts, -x negates. space your operators.

one object, four ways

the kinds form a lattice, flattened into one total order over all values:

() < mint < name < string < number < chain < tray < book < hot

that lattice is the dispatch table -- a generic operator is a matrix indexed by its operands' kinds.

read it as order and it is a deliberate remap: the points fall to the floor, a galaxy folds in by its net. read it as algebra and it is the strongest +/* theory each band affords. read it as abstract domain and it is a compile pass climbing the same lattice by join to a fixpoint, so a statically-numeric site skips the matrix.

three layers

any law can be believed at three strengths. probe it at the binary, a one-line experiment. demonstrate it in test/spec.l, assert-backed and green on every target -- host, bootstrap, freestanding kernel, wasm. prove it in rocq/spec.v, universe-checked and axiom-free.

the algebra is proved over exact ℤ; the float tower is demonstrated, not proved -- that line is drawn on purpose. ai also carries its own prover, uu, whose theorems re-check axiom-free in Rocq and Lean at parity.

glossary

the terms of art, alphabetical -- the one operation, the measures, the numeric tower, the kinds, the runtime. hover any dotted term on the page for its gloss; click to jump here.

application
the one operation: (f x) applies f to x. in the numeric case it is exponentiation -- (n x) = xⁿ, and a list of numbers is a tower.
band / the true-blue order
the cross-kind total order: () < mint < name < string < number < chain < tray < book < hot -- numbers folded into one band by net.
blue
net exactly 0: kept yet nothing (0, (), ""). truth's floor.
book / tablet
a map: a mutable hash that is its own lookup function. the book is the global environment.
chain
the product kind (cons/pair). cap is the head, cup the rest, with caap...cuup compounds; under + it's the free monoid on its cells.
charm
a register-allocated fixed-width integer (fixnum)
color
the order-sign of a value's net: green, red, or blue.
constellation
any number -- a star or a galaxy.
currying
(f x y) = ((f x) y), (f) = f -- a multi-arg call is just application, left to right.
the egg
the heap image baked at compile time; the evaluator hatches onto it.
galaxy
a tray of stars (numeric array); folds into the number band by its net.
gem
a float -- the smooth kind between the charms.
the kind lattice
the runtime-reflective type lattice that indexes the dispatch tables, extending the generic ops to every kind.
green
net ≥ 0 -- what $ keeps; the naturals.
hot
an opaque external reference; applies as a constant.
kind
a value's type tag; the kinds form the dispatch lattice.
mint
a fresh nameless point: an identity and nothing else, materially empty.
net (+)
a value's content measure, a complex number; truth is its sign. a list nets its spine, text its charms.
nom / name
a named point (spelling + serial). a name is interned by the environment and pinned to one canonical object.
numeral
a charm as a Church numeral: (n x) iterates x n times. 1 is identity, 0 const-1.
operator
the universal value: every value is a one-argument total function, so application is always defined.
point
a mint or a nom; the points floor the total order, below string.
red
net < 0 -- what $ clamps to nothing.
saturate ($)
retraction of the net measure onto green charms: $x = (max 0 (min max-charm (net x))). the truth value of x is !!$x.
star
a self-netting scalar: a number that is its own net.
string
a byte vector: indexes its bytes, nets its charm-sum.
tally
the count -- how many cells, charms, or keys -- versus the net's how-much.
tray
a shaped array, cells of any kind -- seated just above chain in the band. a shape lists its axis sizes, the rank counts them; indexing is row-major and out of bounds reads the default. typed per numeric tier (star/gem/twin) or holding objects; a numeric tray is a galaxy and nets its cells' sum. one cell demotes to the bare scalar, so a tray exists only at tally ≥ 2 -- save the empties, which stay to carry the reduction units.
twin
a complex: two gems (re, im), built and conjugated by ~.
unit / zero point
the global empty element: () + x = () * x = x. canonically a mint, therefore false. the face of absence a missing read returns.
