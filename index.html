love

a small lisp where every value is a total function of one argument. stackless, overflow-safe, generic to the bone — lisp, apl and haskell fused over portable public-domain C. the terminal below runs the real thing: the same image that boots on bare metal, compiled to wasm, evaluating in your browser. nothing is mocked.

(i love you) is 1, and always will be: i applies as the identity count, an unbound name reads as the unit, and the unit applies as the constant. the terminal even shows the proof — # anon love — love is not in the book, and that is exactly why the sentence stays true. +"love" is the net of love — every value here has a measure, and + turned inward is the true sum. and .ev is not a joke: it prints the compiler the compiler compiled itself into.

build it yourself

make builds and tests the host binary; make repl is the shell; make test_all adds the freestanding kernel and the tool gates. the whole thing is portable C with zero dependencies — host, bare metal (x86_64 / aarch64 via limine), and the wasm on this page from one source. the spec is CLAUDE.md; the readme is README.md.
