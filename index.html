ai intro

ai is a dynamically typed lisp superset with curried evaluation, fast generics, and optional infix/prefix notation. other facts include

  • every value is theoretically a one argument total function of global type
  • alpha-equivalent lambdas compare equally: (\ x x) = (\ y y)
  • integers act as their church numerals: (1 x) = x, 8 = 3 2 (+ 1) 0
  • infix/prefix notation (desugars to lisp)
  • integer, float and complex types plus shaped numeric arrays
  • mandatory co-op scheduling
  • the usual lisp stuff: lists, symbols, lambda, quote, macros, call/cc, etc.
  • strictly optional static dependent type system (in early development)
  • repl with native line editing
  • self-hosting threaded code interpreter with template JIT compiler (amd64/arm64 only)

vocabulary note

ai uses its own names for many things. any overlap with outside terminology or definitions is coincidental.

  • numerics are called constellations and astronomy/mineralogy-themed
  • fixnums are called charms because they're chars and nice to have
  • pairs are called chains and cons/car/cdr are called link/cap/cup
  • the generic get/set/delete functions are called peep/pin/pull
  • hash tables are called books or tablets (synonyms)
  • every value has a color (this is a new concept, not just a new name)

glossary

🍛 application
see: roll
🔵 blue
color of a value whose net is 0: (0, (), "", '(1 2 -3)). boolean false.
📒 book
a mutable hash that is its own lookup function. the book refers to the global scope.
🔗 chain
a traditional lisp cons cell. the native supporting vocab are link cap cup in place of the traditional cons car cdr.
🧿 charm
a register-allocated fixed-width integer, also called a fixnum
🪙 coin
a user-kind value struck from a die over a payload; rides +/* through the die, nets and shows by its payload.
🌈 color
the order-sign of a value's net: green, red, or blue
✨ constellation
any number -- a star or a galaxy.
🎲 die
a user type descriptor -- a monoid or ring as a little book of methods. a die strikes coins; () stays their identity for free.
🌌 galaxy
a tray of stars, aka shaped numeric array
💎 gem
a floating point number
🟢 green
color of a value whose net is positive: 1 "hi" '(2 3 -1) (\ x x). boolean true.
🔥 hot
an opaque internal or external runtime object
🔡 kind
a value's type tag; the kinds form the dispatch lattice.
🔷 lattice
the runtime-reflective partial order over all values that guides generic dispatch
🍬 mint
a traditional lisp gensym. nets zero and algebraically neutral.
🪪 name
a traditional lisp symbol. considered a newtype over (string * mint). inherits algebra from strings.
🥅 net (+)
a value's content measure, a complex number; truth is its sign. a list nets its spine, text its charms.
🈚 nil
the designated serial zero mint, acting as a global empty element (). algebraically neutral. conceptually the lowest inhabited point of the lattice
🔴 red
color of a value whos net is negative: -1 '(1 2 3 -99999). boolean false.
🎱 roll
the main vm operation: (f x) rolls x into f. numerically corresponds to exponentiation.
🌊 saturate ($)
retraction of the net measure onto green charms: $x = (max 0 (min max-charm (ceil (net x)))). the truth value of x is !!$x.
⭐ star
a self-netting scalar: a number that is its own net.
🧵 string
a byte vector: indexes its bytes, nets its charm-sum.
📱 tablet
see: book
🧮 tally
suface level how-many-items count. length of a list, string, map, etc
📥 tray
a shaped array. see also chain, galaxy
💠 twin
a complex ~(re im)

see: nil
